# Porte Logiche

Le **porte logiche** identificano un particolare canale utilizzabile per la connessione. 

Vanno da 0 a 65535 e sono divise nel seguente modo : 
- **0-1023** (*Well Know Ports*) : sono porte riservate a scopi particolare, come protocolli ecc.
- **1021-49151** : porte registrate che possono essere utilizzare dai client.
- **49151-65535** : porte per uso sperimentale.
